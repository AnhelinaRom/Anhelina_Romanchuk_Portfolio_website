My Projects


Mystical Photography

My first venture into web development involved creating a one-page box layout website, which was crucial in learning the basics of web design. This project taught me how to manage links, integrate images, and utilize empty space effectively.

Cherry Blossom

In my first group project, we developed a sophisticated three-page website designed to sell dolls. This project was not only an opportunity to apply HTML and CSS but also a significant learning experience in teamwork and collaborative coding.

The website's design emerged as both elegant and user-friendly, showcasing a seamless blend of functionality and aesthetics. As the project required collaboration among team members, I gained valuable insights into the dynamics of working with others. One key lesson was understanding that each developer brings a unique approach to coding. This variation necessitated effective communication and the ability to interpret and integrate different coding styles.

While the technical aspects of HTML and CSS presented no major challenges, the primary difficulty lay in harmonizing diverse opinions and preferences. It was crucial to listen to all team members' inputs and find a consensus that met everyone's expectations. This collaborative process underscored the importance of flexibility and compromise in achieving a successful outcome.

To Do List

After investing considerable time in learning JavaScript, I embarked on my first project that combined HTML, CSS, and JavaScript. The project aimed to create a practical tool for personal use: a fully functional To-Do List application.

This To-Do List application allows users to add and manage tasks efficiently. It features functionality for tagging tasks, which enhances organization and usability. Despite finding JavaScript relatively approachable, the development process was both enjoyable and enlightening. I appreciated the opportunity to apply my knowledge in a real-world context and to see the tangible results of my efforts.

The project was a rewarding experience that demonstrated the power of JavaScript in creating interactive and dynamic web applications. I am enthusiastic about revisiting the To-Do List application in the future to further enhance its features and refine its functionality.
